Критерии оценки учебных проектов

Для учащихся 14-16 лет, курс "Основы веб-разработки"

1. Документация и планирование (макс. 20 баллов)

1.1. Четкость формулировок (10 баллов)

  • Суть проекта ясно описана (4 балла)
  • Определены цель и целевая аудитория (3 балла)
  • Название проекта соответствует содержанию (3 балла)

1.2. Полнота описания (10 баллов)

  • Заполнены все разделы документации (4 балла)
  • Подробно описана структура страниц (3 балла)
  • Указаны используемые технологии (HTML, CSS и др.) (3 балла)

2. Техническая реализация (макс. 35 баллов)

2.1. Корректность кода (20 баллов)

  • Валидный HTML-код (5 баллов)
  • Грамотное использование CSS (5 баллов)
  • Отсутствие критических ошибок (5 баллов)
  • Оптимизированная структура файлов (5 баллов)

2.2. Функциональность (15 баллов)

  • Все ссылки работают корректно (5 баллов)
  • Навигация между страницами логична (5 баллов)
  • Контент отображается без искажений (5 баллов)

3. Дизайн и юзабилити (макс. 25 баллов)

3.1. Визуальное оформление (15 баллов)

  • Гармоничная цветовая схема (5 баллов)
  • Удобочитаемые шрифты и контраст (5 баллов)
  • Единый стиль всех элементов (5 баллов)

3.2. Удобство использования (10 баллов)

  • Интуитивно понятная навигация (5 баллов)
  • Логичное расположение контента (5 баллов)

4. Креативность и доработки (макс. 20 баллов)

4.1. Оригинальность решения (10 баллов)

  • Нестандартный подход к реализации (5 баллов)
  • Наличие уникальных элементов дизайна (5 баллов)

4.2. Работа над ошибками (10 баллов)

  • Учтены замечания преподавателя (5 баллов)
  • Виден прогресс между версиями проекта (5 баллов)

5. Дополнительные критерии (макс. +15 баллов)

  • Использование JavaScript (+5 баллов)
  • Применение CSS-анимаций (+3 балла)
  • Собственный дизайн графических элементов (+3 балла)
  • Нестандартный способ публикации (например, GitHub Pages) (+2 балла)
  • Дополнительные страницы сверх требований (+2 балла)

Шкала оценки

  • 90-110 баллов - Отличный результат
  • 70-89 баллов - Хороший уровень
  • 50-69 баллов - Удовлетворительно
  • Менее 50 баллов - Требует доработки
